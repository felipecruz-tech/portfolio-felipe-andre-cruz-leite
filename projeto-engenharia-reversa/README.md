# 🔄 Projeto de Engenharia Reversa com IA (Google AI Studio)

## 📝 Descrição do Projeto
Este projeto explora a prática de **Engenharia Reversa** assistida por Inteligência Artificial, utilizando o **Google AI Studio** como ferramenta central. O objetivo principal não foi apenas a replicação técnica ("clonagem") de uma aplicação existente , mas sim uma análise crítica sobre como a IA redefine o papel do desenvolvedor júnior no mercado atual.

O projeto foca na transição do "saber programar linha por linha" para o "saber resolver problemas", priorizando a visão sistêmica e a comunicação eficaz com modelos de linguagem.

* **Plataforma Base:** [Google AI Studio](https://aistudio.google.com/).

## 📊 Resultados e Aprendizados
O desenvolvimento deste projeto permitiu identificar competências essenciais para o novo cenário da engenharia de software:

* **Pensamento Sistêmico:** A percepção de que é fundamental entender como as partes de um código se conectam para conseguir ajustar e validar o que a IA gera.
* **Prompt Engineering:** O aprendizado de que prompts claros, organizados e modulares ("parte por parte") são cruciais para evitar resultados incompletos.
* **Pensamento Crítico:** A validação constante é necessária para não se tornar dependente da ferramenta, garantindo a integridade do código final.

---

## ⚖️ Reflexão Ética e Integridade
O projeto aborda a linha tênue entre inspiração e plágio no uso de IAs:
* **Legitimidade:** A engenharia reversa é considerada válida para aprendizado quando há mudanças relevantes no design, estrutura ou funcionalidades.
* **Transparência:** O projeto defende a clareza sobre o uso de ferramentas de IA, especialmente em contextos acadêmicos.
* **Originalidade:** Conclui-se que a IA facilita o processo, mas a evolução real do profissional depende da sua capacidade de adaptar e melhorar as ideias geradas.
